# Rückwärtsrechnen

Ein Konsolenprogramm zur Berechnung von Listenpreisen

## Tests ausführen

### Umgebung erstellen

Unter windows, falls venv nicht existiert

```shell
python -m venv venv
Set-ExecutionPolicy -Scope Process Unrestricted
.\venv\Scripts\activate
pip install pytest
```

### Umgebung verwenden (windows)

Unter windows, falls venv nicht existiert

```shell
Set-ExecutionPolicy -Scope Process Unrestricted
.\venv\Scripts\activate
pytest
```
